# News Hub Client

## Prerequisites

- **Node.js** v20.9.0
- **npm** 10.1.0

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/your-username/your-repo.git
    ```

2. Navigate to the project directory:

    ```sh
    cd your-repo
    ```

3. Install dependencies:

    ```sh
    npm install
    ```

## Development Server

To start the development server, run the following command:

```sh
npm run dev
```
